# Your Company Lead-Generation Website

A polished MVP lead-generation website for a fictional small business named Your Company.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploy

This project is ready for Vercel. The build command is `npm run build` and the output directory is `dist`.
